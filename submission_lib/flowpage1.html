Simple submission to {{ crrt_conf.name }}

  1. Using this page you can create a speaker profile. (If you have used this system before you existing details will be shown.)
  2. Next you will be asked to give details of your talk.
  3. Finally you can submit the talk to {{ crrt_conf.name }}.

Co-speakers: If your submission has more than one presenter then: one person should make the submission. This person will be the main contact. At the end of the sumission process you can add co-speakers.

If you are submitting on behalf of someone else then please make a submission and add them as a co-speaker. Make a note in your own "speaker bio" that you are the contact but not the speaker.

If you wish to edit an existing submission, speaker or talk details, or you wish to review feedback, then you will need to log into the main Mimas system.

All submissions are indexed using the e-mail address provided so please use the same email address for logging in.

If you are making more than one submission to the same conference you will find it less work to login and use the main system where you can save some repeated data entry.
